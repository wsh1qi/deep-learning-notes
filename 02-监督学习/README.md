# 02 · 监督学习

## 二分类

### 逻辑回归（Logistic Regression）

目标：力求每次预测正确。

**Sigmoid 函数**：

$$\sigma(z) = \frac{1}{1 + e^{-z}}$$

$$\sigma'(z) = \sigma(z)(1 - \sigma(z))$$

> 缺点：容易出现**梯度消失**。

**二分类交叉熵损失**：来自极大似然估计（MLE），Sigmoid 输出值连乘后取对数推导而来。

### 支持向量机（SVM）

思想：**最大化分类间隔**。

- 线性不可分时：使用**软间隔**。
- 数据分布非线性时：使用**核方法**升维。

## 多分类

### 多分类逻辑回归

为每个类别 $c$ 维护一个线性超平面。给定 $(x, y)$，使用 Softmax 函数输出各类别后验概率。

### 多分类 → 二分类转换
- 一对多（One-vs-All）
- 一对一（One-vs-One）

---

## 类别不平衡

当类别分布不均匀时，提高小类重要性：

- **样本重加权（Re-weighting）**
- **样本重采样（Re-sampling）**
