# 附录 · 数学公式速查

## 常见激活函数及导数

| 函数 | 表达式 | 导数 |
|------|--------|------|
| Sigmoid | $\sigma(z) = \frac{1}{1+e^{-z}}$ | $\sigma'(z) = \sigma(z)(1-\sigma(z))$ |
| Tanh | $\tanh(z) = \frac{e^z - e^{-z}}{e^z + e^{-z}}$ | $\tanh'(z) = 1 - \tanh^2(z)$ |
| ReLU | $\max(0, z)$ | $0$ if $z<0$, $1$ if $z>0$ |

## 常见损失函数

| 损失函数 | 公式 | 适用场景 |
|----------|------|----------|
| MSE | $\frac{1}{n}\sum(y - \hat{y})^2$ | 回归 |
| BCE | $-[y\log\hat{y} + (1-y)\log(1-\hat{y})]$ | 二分类 |
| CE | $-\sum y_c \log \hat{y}_c$ | 多分类 |
| Dice Loss | $1 - \frac{2\|A \cap B\|}{\|A\| + \|B\|}$ | 分割 |

## 泛化误差收敛速率

| 假设空间 | 速率 |
|----------|------|
| 有限可分 | $O(1/m)$ |
| 有限不可分 | $O(1/\sqrt{m})$ |
| VC 维 = $k$ | $O(\sqrt{k/m})$ |
