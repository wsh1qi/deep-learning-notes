# 03 · 半监督与无监督学习

## 无监督学习

### K-means 聚类

痛点：
- 容易陷入局部最优
- 容易被噪声点干扰

**改善方法**：K-中位数算法（K-medians），重新选点时取中位数而非算术平均值。

---

## 半监督学习

### 自训练（Self-training）

1. 用少量带标签数据训练分类器
2. 用分类器预测无标签数据
3. 找出 Top-$k$ 置信度最高的无标签样本，打上伪标签
4. 移入有标签数据集
5. 重新训练，迭代

> 问题：容易出现**误差传播**（error propagation）。

### 协同训练（Co-training）

两个分类器互相给对方数据，利用不同"视角"互补。

### 主动学习（Active Learning）

选择最有价值的无标签数据打标签：

| 策略 | 思路 |
|------|------|
| 不确定性采样 | 挑选最不确定的样本打标签 |
| 期望误差缩小 | 挑选对大局贡献最大的样本 |
